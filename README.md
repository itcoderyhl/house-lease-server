## 【原创】springboot+vue前后端分离的房屋出租网（源码+文档）

## 系统介绍

基于SpringBoot+Vue前后端分离的房屋出租网（源码+文档），系统包含管理员、房东、用户角色，功能如下：
管理员：首页统计（房屋统计、合同统计）；用户管理；房东管理；房屋管理（房屋分类、房屋管理）；租赁管理（消息信息、预约看房、合同管理、房屋评价）；论坛管理（板块分类、帖子管理、评论管理）；行业资讯；系统留言；系统公告；个人信息；密码修改。
房东：注册；登录；房屋管理（查看房屋分类、房屋管理）；租赁管理（消息信息、预约看房、合同管理、房屋评价）；系统留言；修改个人资料；修改密码。
用户：注册、登录系统；首页；房屋出租（房屋分类、房屋列表、详情、在线咨询、收藏、预约看房、评价）；社区交流（论坛版块、帖子列表、详情、点赞、评论）；行业资讯；系统留言；公告信息。用户还具有个人中心功能，功能包括：消息咨询、预约看房、租赁合同（合同确认、下载、终止）、房屋评价、房屋收藏、我的帖子、帖子评论、帖子点赞、系统留言、个人资料、修改密码。
系统界面美观大方，功能丰富，使用了SpringBoot、Vue、ElementUI、Echarts等技术栈，适合作为毕业设计、课程设计，有需要可提供指导。

项目由本人原创开发，经过多次测试，项目运行无问题，请放心使用，作者QQ：1344127185。

## 使用技术

服务器端技术栈：

- Springboot
- SpringMvc
- Mybatis
- MySQL

前端技术栈：

- vue
- elementUI
- axios
- echarts

## 环境介绍

- IDEA集成环境
- JDK1.8
- MySQL5.6+

可提供所有运行软件、远程安装、技术指导。
