# house-lease-server
基于SpringBoot+Vue前后端分离的房屋出租网（源码+文档），系统包含管理员、房东、用户角色，功能如下： 管理员：首页统计（房屋统计、合同统计）；用户管理；房东管理；房屋管理（房屋分类、房屋管理）；租赁管理；论坛管理（板块分类、帖子管理、评论管理）；行业资讯；系统留言；系统公告；个人信息；密码修改。 房东：注册；登录；房屋管理；租赁管理；系统留言；修改个人资料；修改密码。 用户：注册、登录系统；首页；房屋出租（房屋分类、房屋列表、详情、在线咨询、收藏、预约看房、评价）；社区交流（论坛版块、帖子列表、详情、点赞、评论）；行业资讯；系统留言；公告信息。用户个人中心，功能包括：消息咨询、预约看房、租赁合同、房屋评价、房屋收藏、我的帖子、帖子评论、帖子点赞、系统留言。
